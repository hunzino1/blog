Rails常用命令和Rake
==============================

该文档列出了 Rails常用命令和Rake 的基本知识点.

阅读完该文档后，您将会了解到:

* Rails常用命令
* Rake 主要内容概括

--------------------------------------------------------------------------------

Rails常用命令
---------------------
- rails console
- rails server
- rake
- rails generate
- rails dbconsole
- rails new app_name
- rails destroy

Rake用法
-----------------------
- Rails常用rake命令
  + rake --tasks
  + rake db: + ...
  + rake routes

- 自定义rake
  + :namespace
  + 传入参数， args
