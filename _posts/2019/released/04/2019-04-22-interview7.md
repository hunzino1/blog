### 18 给一个可以随机等概率生成1-7之间的函数f7,如果构造可以随机等概率生成1-10的函数f10

博文中的2位N进制数的意思：

rand7 就是7进制数，[1 - 7]

7是进位，故 7 * (rand7 - 1) + rand7 就是两位7进制数， 这样产生的数一定是随机数。

[1](https://www.cnblogs.com/youxin/p/3351216.html)

```ruby
def rand10()
{
    int x = 0;
    do
    {
        x = 7 * (rand7() - 1) + rand7(); [1, 49]
    }while(x > 40);
    return 1 + x%10;
}
```
